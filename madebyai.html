How This Was Made

This entire project was made by AI. A human was also on hand, providing the same critical contribution to the project as a lamppost does to the dawn.

Code Generation

Every line of HTML, CSS, and JavaScript for the landing, reservation, and drink order pages was written by a large language model. Through a series of prompts and iterative feedback, we built the layout, styling, and functionality, including the interactive calendar events and modals.

Backend & Data

The reservation system is powered by a **Google Sheet** that acts as a simple database. The website communicates with the spreadsheet through a **Google Apps Script**, which was also written by AI. This allows for real-time seat availability updates without a complex database setup.

Image & Asset Creation

The core visual assets—including the theater logo, ornate poster frame, and the images of popcorn, candy, and drinks—were generated by an AI image model. However, AI-generated images often require manual adjustments. The open-source tool **GIMP** was used for post-processing tasks, such as making the center of the frame transparent.

AI-Assisted Learning

Learning new software like GIMP was accelerated by using **Google's AI console** as a live tutor. By sharing the screen and using voice chat, the AI could provide real-time, step-by-step instructions, turning a potentially complex editing task into a guided learning session.
